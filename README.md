# OSS-Vision 転送用サーバー (web)

このリポジトリは、ネイキッドドメイン（`https://oss-vision.com`）へのアクセスを、公式Webサイト（`https://www.oss-vision.com`）へ自動転送（リダイレクト）するために稼働している専用リポジトリです。

## 💡 なぜこの仕組みが必要なのか？

Google Workspace や Google サイトの標準転送機能では、**`https://`（SSL/暗号化通信）でのネイキッドドメイン転送がサポートされていない**という仕様があります。
そのため、ブラウザが自動的に `https://` でアクセスを試みた際、お客様がエラー画面に遭遇してしまう問題がありました。

この問題を解決するため、GitHub Pages が提供する「無料のSSL証明書（HTTPS対応）」を利用し、アクセスを一度ここで受け止めてから、即座に公式の `www` ありURLへ転送する仕組みを構築しています。

## ⚙️ 動作の仕組み

1. **DNS（お名前.com）:** `oss-vision.com` の Aレコード を GitHub Pages の IPアドレス に向けています。
2. **GitHub Pages:** カスタムドメインとして `oss-vision.com` を設定し、「Enforce HTTPS」で安全な通信を確立しています。
3. **リダイレクト:** このリポジトリにある `index.html` の `<meta http-equiv="refresh">` タグによって、0秒で公式Webサイトへ転送されます。

## ⚠️ メンテナンス時の注意

* `index.html` は転送の要となる重要なファイルです。**誤って削除しないようにしてください。**
* （※かつてこのリポジトリにあった OSS-Vision の会社概要やコンセプト原稿などの古いテキストは、Gitのコミット履歴からいつでも復元・参照が可能です。）
